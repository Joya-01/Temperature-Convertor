# Temperature Converter 

A simple and interactive web application that allows users to convert temperature values between different units (Celsius, Fahrenheit, and Kelvin) instantly.

## Project Overview
This project is a lightweight web tool built with standard web technologies. It provides a user-friendly interface where users can input a temperature in one unit and immediately see the corresponding values in other units.

## Technologies Used
- **HTML5**: For the structure of the webpage.
- **CSS3**: For styling and layout (`style.css`).
- **JavaScript**: For the conversion logic and interactivity (`index.js`).

## How to Run
You can run this project directly in your browser without any installation.

1. **Clone the repository:**
   ```bash
   git clone [https://github.com/Joya-01/Temperature-Convertor.git](https://github.com/Joya-01/Temperature-Convertor.git)

2. **Navigate to the folder:**
   ```bash
   cd Temperature-Convertor

3. **Open the application:**

  Double-click the index.html file to open it in your default web browser.

   OR, if using VS Code, you can use the "Live Server" extension.
